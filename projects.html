Project Overview

Our project aims to design a healthcare management system leveraging algorithms to solve problems efficiently. Each team member contributed to a specific sub-project, tackling challenges with real-life applications of algorithms.

Team Contributions

  • Sub-Project 1: Healthcare Resource Allocation System using Dijkstra's Algorithm.
  • Sub-Project 2: Emergency Incident Categorization with Quick Sort and Rabin-Karp.
  • Sub-Project 3: Patient Scheduling using Priority Queue.
  • Sub-Project 4: Inventory Management with Hashing.
